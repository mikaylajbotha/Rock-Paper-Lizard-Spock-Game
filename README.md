
# Rock Paper Scissors Lizard Spock

A Java GUI game to play **Rock Paper Scissors Lizard Spock** against the computer. The game shows choices, results, and keeps scores.

---

## Features

- GUI built with **Java Swing**
- 5 choices: Rock, Paper, Scissors, Lizard, Spock
- Interactive buttons with **icons and hover effects**
- **Score tracking** for player and computer
- **Color-coded results**:
  - Green = Player wins
  - Red = Computer wins
  - Orange = Tie
- Scrollable **rules panel**

---

## Game Rules

- Rock crushes Scissors  
- Scissors cuts Paper  
- Paper covers Rock  
- Rock crushes Lizard  
- Lizard poisons Spock  
- Spock smashes Scissors  
- Scissors decapitates Lizard  
- Lizard eats Paper  
- Paper disproves Spock  
- Spock vaporizes Rock  

---

## Requirements

- Java JDK 8+  
- Java IDE or editor  
- Optional: Icons for buttons in `images/` folder
